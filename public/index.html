VBA helpers API

Open repository

API routes

  • GET /api/helpers - получение хелперов
  • GET /api/helpers/search-by-category/:keyword - поиск хелперов по категории
  • GET /api/helpers/search-by-name/:keyword - поиск хелперов по имени
  • GET /api/helpers/search-by-title/:keyword - поиск хелперов по заголовку
  • GET /api/helpers/search-by-keyword/:keyword - поиск хелперов по ключевым словам (фразе)
  • GET /api/categories - получение списка категорий
  • GET /api/categories/search/:keyword - поиск категории по имени
  • GET /api/categories/search-by-keyword/:keyword - поиск категорий по ключевым словам (фразе)
